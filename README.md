# SA Lottery Predictor

A lightweight predictor for South African lotteries using group pattern analysis. Generates affordable ticket sets (6 and 13 lines) for each lottery.

## Supported Lotteries
- Daily Lotto (5/36)
- Daily Lotto Plus 1 (5/36)
- Lotto (6/52 + bonus)
- Lotto Plus 1 (6/52 + bonus)
- Lotto Plus 2 (6/52 + bonus)
- Powerball (5/50 + bonus)
- Powerball Plus (5/50 + bonus)
- Sportstake (6 matches)
- Sportstake 8 (8 matches)
